## Descrição do Problema
O projeto da Smart Home tem como objetivo desenvolver um sistema capaz de gerenciar dispositivos inteligentes em uma residência, permitindo o controle dos dispositivos principais de uma casa, como as lâmpadas, ar condicionado, portão e aparelhos de som. A proposta envolve a simulação de uma casa automatizada, onde dispositivos são controlados remotamente por meio de uma interface textual, além de permitir a criação de automações personalizadas e monitoramento do estado dos equipamentos.  

O sistema busca representar o funcionamento de ambientes inteligentes, abordando conceitos como modularidade, comunicação entre componentes e gerenciamento de estados. Além disso, também são consideradas situações de erros que permitem a detecção de comportamentos anormais e o registro de eventos, contribuindo para uma simulação mais real.  

O projeto foi realizado na linguagem C++, tendo paradigma em orientação a objetos, com auxílio da ferramenta Makefile para compilação.  

## Objetivos
- Implementar um sistema de gerenciamento de dispositivos inteligentes;  
- Permitir controle individual e integrada dos dispositivos;  
- Desenvolver automações por meio de macros;  
- Simular monitoramento contínuo e detecção de falhas;  
- Aplicar conceitos de orientação a objetos (herança, composição e polimorfismo);  
- Estruturar o sistema de forma modular e escalável.  

## Motivação
A escolha da Smart Home como tema foi motivada pela necessidade da automatização de sistemas no contexto atual, devido ao aumento do uso de dispositivos autônomos e interconectados para facilitar o controle geral de uma casa. Nesse projeto, é possível aplicar os conceitos de programação estudados em sala em um problema próximo da realidade, permitindo explorar tanto aspectos técnicos quanto de modelagem de sistemas. Além disso, possibilita trabalhar com diferentes níveis de abstração, desde a interação com o usuário até o controle interno dos dispositivos.  

## Funcionalidades Principais
- Controle de dispositivos (ligar, desligar, ajustar parâmetros);  
- Execução de macros personalizados;  
- Monitoramento contínuo dos dispositivos;  
- Detecção e registro de falhas;  
- Exibição de estado em tempo real via interface textual.
